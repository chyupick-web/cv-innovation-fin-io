Airplane mode wallet

securely send Bitcoin and Ethereum

  • 1
    Generate invoice in Kassa

    Select the recipient address, currency and the amount to send

  • 2
    Scan QR with app to receive transaction data

    App will show the transaction details and sign them if you confirm

  • 3
    Send back signed transaction

    Show QR to Kassa which will broadcast the transaction to Blockchain

Features

Open Sourced

Of course, flight.wallet is opensourced. Visit our GitHub to make your own security audit.

One mnemonic

We will create all private keys using only one mnemonic. You can restore all wallets in case of device lost.

Secured

No data will left your device via Internet. All keys will be encrypted and safe and sound.
